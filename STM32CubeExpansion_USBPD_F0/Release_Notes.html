Release Notes for USB Type-C™Power Delivery (PD) on software expansion for STM32Cube 

Copyright 2017 STMicroelectronics

 


USB Type-C™ is the newest USB connector ecosystem, it addresses the evolving needs of platform and devices, while retaining the functional benefits of  USB.
STM32 USB-PD package consists of a library (in a binary format) and application examples for STM32F0 devices acting as USB Power Delivery controllers, and will be extended to other STM32 Series in the future.
This package includes examples covering most of applicative use cases and allowing to develop applications based on USB-PD (Provider, Consumer, Dual Role Port, Dual Port management, VDM feature support). The library and associated examples can be used with STM32F0 devices and a dedicated P-NUCLEO-USB001 shield containing analog front-ends and USB Type-C™ connectors.
     
         

The  X-CUBE-USB-PD software expansion for STM32 structure is layered on top of the HAL abd BSP drivers allowing a modular and flexible usage of the different modules. It can easily be tailored to different hardware platforms.

Supported Devices and boards

  • STM32F072RB-Nucleo RevC board with built-in STLink with following configuration:
    • Solder bridges SB48, SB49, SB62, SB63 are closed
    • Solder bridges SB13, SB14, SB15, SB21 are open
    • 0 Ohm resistors R34, R36 are removed
  • P-NUCLEO-USB001 shield, MB1257, RevB or RevC.

Main Changes

  • Main changes
    • Optimization of the SRAM usage for all examples to be below 8KB.
    • In  STM32 USB-C Power Delivery Library:
      • Allow DPM to select the profiles dynamically when running as a consumer
      • Add new function to request new profile from DPM
      • Add new function to set Sink power request from DPM
      • Export received profiles to DPM
    • Add new USB-C Power Delivery Provider application running in standalone mode without RTOS
